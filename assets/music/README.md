# Folder Musik Latar

Taruh file musik instrumen bebas royalti di sini untuk diputar sebagai musik latar video TikTok.

## Format yang didukung:
- `.mp3`
- `.wav`
- `.ogg`
- `.m4a`

## Rekomendasi sumber musik bebas royalti:
- [Pixabay Music](https://pixabay.com/music/) — Filter: Cinematic / Dramatic
- [Free Music Archive](https://freemusicarchive.org/)
- [YouTube Audio Library](https://studio.youtube.com/channel/music)

## Tips:
- Pilih musik instrumental (tanpa vokal) genre **cinematic dark / ambient / dramatic**
- Durasi minimal 30 detik agar bisa di-loop
- Sistem akan otomatis memilih musik secara acak jika ada lebih dari 1 file
- Volume musik akan otomatis dikecilkan ke 10% (≈ -20dB) agar tidak menutupi suara TTS
